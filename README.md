## Getting Started

### Installation

1. Clone the repo
```sh
git clone https://github.com/pedrovsiqueira/portfolio-pedro-siqueira
```
2. Install Yarn packages
```sh
yarn add
```
3. Run the app
```sh
yarn start
```
